# Windows optional extras — disk image support
#
# Use pip extras (preferred):
#   pip install ".[windows]"
#
# This installs:
#   pytsk3           — raw / E01 / VMDK / VHD image parsing (The Sleuth Kit)
#   windowsprefetch  — MAM decompression for Windows 10/11 prefetch files
#
# For E01 images, Arsenal Image Mounter is often simpler than pytsk3:
#   https://arsenalrecon.com/products/arsenal-image-mounter
#   Mount the image as a drive letter, then:
#   supertimeline run E:\ -o out.parquet
